# 🎨 Media Kit

This Media Kit contains official brand assets for partners, community members, creators, media outlets, and Corporate Survivors worldwide.

> **Please use official assets at all times.** Consistency builds trust — and trust is the one KPI we actually care about.

## 🖼️ Logo Assets

All logo files are located in the [`/logos`](./logos/) folder.

### Primary Logo — Hexagon + Cubert + Wordmark

The full brand mark. Use this as the default in any context where space allows.

**File:** `cubert-coin-primary.png`

**Recommended usage:**
- Website headers
- GitHub repository
- Official documents and press materials
- Partnership announcements

### Icon — Cubert only

The mascot alone, without the wordmark. Ideal for small formats.

**File:** `cubert-coin-icon.png`

**Recommended usage:**
- Social media profile pictures
- Community platform avatars
- Favicons

### Wordmark — Typography only

The official Cubert Coin logotype without the character.

**File:** `cubert-coin-wordmark.png`

**Recommended usage:**
- Headers and banners
- Merchandise
- Co-branded materials

## 🔤 Typography

The Cubert Coin wordmark uses custom styling developed exclusively for the project.

**Guidelines:**

- Always use the official logo assets — do not recreate the wordmark manually
- Maintain original proportions and spacing at all times
- Do not substitute brand colors with approximations
- When in doubt, use the asset as provided

## ✅ Usage Guidelines

### Allowed

- Community-created content (memes, posts, videos)
- News coverage and editorial materials
- Educational content about Cubert Coin
- Partnership and collaboration announcements

### Not Allowed

- Modifying, stretching, or distorting any logo
- Altering official colors
- Implying official endorsement without authorization
- Use in misleading, fraudulent, or harmful materials
- Recreating or reinterpreting the Cubert character

> When in doubt, ask in the [Discord](https://discord.gg/UqJwHACWhz) before publishing.
